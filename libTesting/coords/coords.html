Vizra.coords

Get coordinates for shape distribution

Grid

Pass in 'grid', 'iso', 'polar' ('random'?) - screen width and height can be found in class, grid width & height are set in pixels or 'tight', 'even', 'loose', extendEdges = true

new VizraCoords{type, {gridSize or distrib keyword}, extendEdges = true}

This gives you a loose triangular grid with borders

new VizraCoords('grid', [40, 40])

Iso

new VizraCoords('iso', 'loose', false)

Polar

new VizraCoords('polar', 'even')
