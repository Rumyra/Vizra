Parameters:
Name Type Attributes Description
new Canvas * <optional>
 of Canvas type from Canvas.js to allow proper dimensions
type String <optional>
 can be 'square', 'iso', 'polar'. 'size' or 'custom'. 'custom' means an array of coords should be passed into the next argument
or String {Array} [distribution] can be 'tight', 'regular', 'loose' or 'sloppy'. Determines how close the coords should be on the grid. If 'custom' is set as the grid type, an array of coord pairs should be passed in instead. If 'size' is set one dimentional array of [x, y] size must be specified
extend edges Boolean <optional>
 whether the grid should fill up to the edges of the canvas (if default true), or a calculated gap should show if set to false

Example
