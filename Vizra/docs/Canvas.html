Canvas(elementopt, widthopt, heightopt, resizeopt)

Canvas class

Constructor

new Canvas(elementopt, widthopt, heightopt, resizeopt)

In the instance that width & height aren't specifed, one assumes a fullscreen canvas which resets styling on the body & html elements.
Parameters:
Name Type Attributes Default Description
element string <optional>
 'canvas' string representing element: such as 'canvas' or '#screen'. It is optional, if nothing is set one of two things occurs, the instatiation will try to find a canvas element in the HTML, if it doesn't find one it will create one automatically. So for the most part don't bother setting anything! Let the class do the work.
width * <optional>
 CSS.vw(100) sets the viewed width of the canvas, defaults to full width of window. Takes a CSS value type only.
height * <optional>
 CSS.vh(100) sets the viewed height of the canvas, defaults to full height of window. Takes a CSS value type only.
resize boolean <optional>
 true canvas sizes will resize if window is resized, draw will disappear atm though if no rqf is running
